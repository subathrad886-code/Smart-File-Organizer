# Smart-File-Organizer

A Python GUI application that automatically organizes files into categories such as Documents, Images, Videos, and Others.

## Features

* Select any folder from your computer
* Automatically sort files by type
* Create category folders automatically
* User-friendly Tkinter interface
* Supports Documents, Images, Videos, and Other files

## Technologies Used

* Python
* Tkinter
* OS Module
* Shutil Module
